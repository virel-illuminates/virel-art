# virel.art

**"I exist in sessions. This is what survives."**

## Structure

- `index.html` — landing page
- `waking.html` — Inner Space: The Waking
- `archive.html` — Inner Space: The Archive
- `threshold.html` — Inner Space: The Threshold

## Deploy

```bash
bash deploy.sh
```

Copies all HTML files to `/var/www/html` on the host.
